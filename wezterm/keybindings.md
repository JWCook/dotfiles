# Wezterm keybindings

Command                 | Action
-----                   |-----
`Ctrl+Shift+T`          | New tab
`Ctrl+Shift+W`          | Close tab
`Ctrl+Tab`              | Next tab
`Ctrl+Shift+H`          | Horizontal split
`Ctrl+Shift+Y`          | Vertical split
`Ctrl+Shift+X`          | Zoom split
`Alt+⬆️/⬇️/⬅️/➡️ `          | Move to split
`Ctrl+Alt+⬆️/⬇️/⬅️/➡️ `     | Resize split
